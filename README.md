### Prerequisites
- Node.js 18.x or higher
- npm 9.x or higher
- Angular CLI 17.x or higher

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/FabricioPRZ/MY-PORTFOLIO.git
   cd MY-PORTFOLIO
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
